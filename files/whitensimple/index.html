White-n-Simple

A very simple and subtle white theme with no icon backgrounds.

Description:

A very simple and subtle white theme with no icon backgrounds. A current work in progress with 90% of the stock iOS applications included in version 1.0. This theme works well with white backgrounds as well as dark backgrounds. Every icon was created with precision to make sure that all icons fit perfectly with the theme itself.
